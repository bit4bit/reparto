24   # Filter is a way, to process translations: escape HTML entries, convert from
 
25   # Markdown syntax, etc.
 
26   #
 
27   # In translation file filtered content must be marked by YAML type:
 
28   #
 
29   #   filtered: !!custom_type
 
30   #     This content will be processed by filter
 
31   #
 
32   # Filter function will be receive filtered content as first argument, struct
 
33   # with filter config as second and filter parameters as next arguments. You
 
34   # can set passive filter, which will process translation only on loading.
 
35   #
 
36   #   R18n::Filters.add('custom_type', :no_space) do |content, config, replace|
 
37   #     content.gsub(' ', replace)
 
38   #   end
 
39   #   R18n::Filters.add('custom_type', :passive => true) do |content, config|
 
40   #     content + '!'
 
41   #   end

44   # 
45   # Use String class as type to add global filter for all translated strings:

138       # Add new filter for +type+ with +name+ and return filter object. You
 
139       # can use String class as +type+ to add global filter for all translated
 
140       # string.
 
141       # 
142       # Filter content will be sent to +block+ as first argument, struct with
 
143       # config as second and filters parameters will be in next arguments.
 
144       # 
145       # Options:
 
146       # * +position+ – change order on processing several filters for same type.
 
147       #    Note that passive filters will be always run before active.
 
148       # * +passive+ – if +true+, filter will process only on translation
 
149       #   loading. Note that you must add all passive before load translation.
 
150       def add(types, name = nil, options = {}, &block)
 
151         options, name = name, nil if name.is_a? Hash
 
152         types = Array(types)
 
153 
154         unless name
 
155           @last_auto_name ||= 0
 
156           begin
 
157             @last_auto_name += 1
 
158             name = @last_auto_name
 
159           end while defined.has_key? name

161           delete(name)
 
162         end

164         filter = Filter.new(name, types, block, true, options[:passive])
 
165         @defined[name] = filter
